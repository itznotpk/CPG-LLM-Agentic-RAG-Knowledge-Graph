## A) Mechanical Treatment

| Device | When/How to Apply | Precautions | Adverse Events |
|--------|-------------------|-------------|----------------|
| **Vacuum Erectile Devices (VED)** | 5-10 minutes before intercourse | Antiplatelets and anticoagulants - may cause bleeding and bruising | Issues with lack of spontaneity, uncomfortable, sensation of cold penis |
| **Low-intensity Extracorporeal Shockwave Therapy (Li-ESWT)** | 20-30 minutes for each session, once a week for at least 6 sessions | - | Priapism, pain at the site, bleeding and bruising, haematuria, skin infection, painful erection, worsening penile curvature |

## B) Pharmacological Treatment (PDE5 Inhibitors)

### Sildenafil

| Parameter | Details |
|-----------|---------|
| **Initial Dose** | 50 mg taken approximately 1 hour before sexual activity |
| **Effective Duration** | 30 minutes to 4 hours after administration |
| **Half-life** | 4 hours |
| **Dose Range** | 25-100 mg, maximum once daily |
| **Dose Adjustments** | |
| - Renal impairment (CrCl <30 ml/min or HD) | Starting dose 25 mg |
| - Hepatic impairment | Starting dose 25 mg |
| - Geriatric use | Starting dose 25 mg |
| - Alpha-blockers | Starting dose 25 mg |
| - Ritonavir | Maximum 25 mg over 48 hours |
| - CYP3A4 inhibitors | Starting dose 25 mg |

### Tadalafil

| Parameter | Details |
|-----------|---------|
| **As Needed Dose** | 10 mg taken at least 30 minutes before sexual activity |
| **Once Daily Dose** | 2.5 mg once daily without regard to timing of sexual activity |
| **Effective Duration** | Up to 36 hours after administration |
| **Half-life** | 17.5 hours |
| **Dose Range** | As needed: 5-20 mg max once daily; Daily: 2.5-5 mg |
| **Dose Adjustments** | |
| - CrCl <30 ml/min or HD | Maximum 5 mg over 72 hours |
| - Hepatic impairment (Child Pugh A/B) | Maximum 10 mg per day |
| - Hepatic impairment (Child Pugh C) | Not recommended |
| - CYP3A4 inhibitors | Maximum 10 mg over 72 hours |
| - Alpha-blockers | Stable dose prior to initiation; start tadalafil at lowest dose |

### Avanafil

| Parameter | Details |
|-----------|---------|
| **Initial Dose** | 100 mg taken approximately 30 minutes before sexual activity |
| **Half-life** | 5 hours |
| **Dose Range** | 50-200 mg, maximum once daily |
| **Dose Adjustments** | |
| - Renal impairment (HD) | Do not use |
| - Hepatic impairment (Child Pugh C) | Do not use |
| - Strong CYP3A4 inhibitors | Contraindicated |
| - Moderate CYP3A4 inhibitors | Maximum 50 mg over 24 hours |
| - Alpha-blockers | Stable dose prior; start at 50 mg |
| **Time to Nitrate** | 12 hours |

### Vardenafil

| Parameter | Details |
|-----------|---------|
| **Initial Dose** | 10 mg taken approximately 60 minutes before sexual activity |
| **Half-life** | 4-5 hours |
| **Dose Range** | 5-20 mg, maximum once daily |
| **Dose Adjustments** | |
| - HD | Do not use |
| - Hepatic impairment (Child Pugh B) | Starting dose 5 mg, maximum 10 mg |
| - Hepatic impairment (Child Pugh C) | Do not use |

| - Alpha-blockers | Stable dose prior; start at 5 mg |
| - Geriatric (≥65 years) | Starting dose 5 mg |
| **Time to Nitrate** | 24 hours |
| **Note** | Not available in Malaysia |

## Contraindications (ALL PDE5i)

| Contraindication | Details |
|------------------|---------|
| Known hypersensitivity | To PDE5i or any component of tablet |
| Nitrates/Nitrites | Concurrent use with NO donors, organic nitrates or organic nitrites (e.g. glyceryl trinitrate) |
| Riociguat | Administration with guanylate cyclase (GC) stimulators |
| QT prolongation (Vardenafil) | Patients with congenital QT syndrome or taking class IA/III antiarrhythmics |

---

## Adverse Events

| Category | Events |
|----------|--------|
| **Common** | Headache, flushing, dyspepsia, nasal congestion, nasopharyngitis, back pain, myalgia, visual abnormalities |
| **Rare** | NAION (non-arteritic anterior ischemic optic neuropathy) |
| **Serious (Vardenafil)** | QT prolongation |

### Warnings

> **Emergency:** Patients should seek emergency treatment if an erection lasts >4 hours. Use PDE5i with caution in patients predisposed to priapism.

> **Hearing Loss:** Patients to stop taking PDE5i and seek prompt medical attention in the event of sudden decrease or loss of hearing.
